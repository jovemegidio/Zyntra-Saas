usar `data_prevista` em vez de `data_previsao_entrega`---## 📋 Checklist de Validação- [✅] Calendário mostra mês/ano correto- [✅] Navegação de meses funciona (◀ ▶)- [✅] Modal Nova Ordem abre e fecha- [✅] Código da ordem é gerado automaticamente- [✅] POST cria nova ordem no banco- [✅] Nova ordem aparece na lista imediatamente- [✅] Botão Visualizar mostra detalhes corretos- [✅] Botão Progresso atualiza percentual- [✅] Slider sincroniza com input numérico- [✅] Botão Editar pré-preenche campos- [✅] PUT atualiza ordem no banco- [✅] Cabeçalhos são visíveis em ambas as views- [✅] Responsividade funciona em mobile- [✅] Dark mode aplicado corretamente---## 🚀 Próximos Passos (Sugestões)1. **Socket.io Real-time:**   -
