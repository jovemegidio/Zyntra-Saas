**GTIN**: Código de barras opcional (único se preenchido)5. **Descrição**: Descrição detalhada opcional6. **Marca**: Marca do produto opcional7. **Variação**: Variações do produto (cores, tamanhos, Tabela```sqlprodutos (  id INT PRIMARY KEY AUTO_INCREMENT,  codigo VARCHAR(20) NOT NULL UNIQUE,  nome VARCHAR(255) NOT NULL,  variacao VARCHAR(100),  marca VARCHAR(50),  data_criacao
