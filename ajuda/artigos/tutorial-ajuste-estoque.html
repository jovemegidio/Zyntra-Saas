Como Fazer Ajuste Manual de Estoque

Aprenda a corrigir divergências de estoque quando o saldo do sistema não corresponde ao físico.

O ajuste manual de estoque permite corrigir o saldo de um produto diretamente, sem necessidade de um inventário completo. Ideal para correções pontuais de pequenas divergências.

Pré-requisitos

  • Acesso ao módulo de Estoque com permissão de Ajuste
  • Identificação prévia da divergência
  • Justificativa para o ajuste (obrigatória para auditoria)

Passo a passo

Passo 1 — Acessar "Ajustes"

No módulo Estoque, acesse Movimentações e clique em + Nova Movimentação, selecionando o tipo Ajuste.

Passo 2 — Confirmar ajuste

Ao confirmar, o sistema registra a movimentação de ajuste e atualiza o saldo imediatamente.

Auditoria

Todo ajuste manual fica registrado com data, hora, usuário e motivo. Gestores podem consultar o relatório de ajustes para acompanhar divergências recorrentes.

Quando usar ajuste manual vs inventário

SituaçãoRecomendação
1-3 produtos com divergência pontualAjuste manual
Contagem periódica de todo o estoqueInventário completo
Perda identificada durante operaçãoAjuste manual
Auditoria ou balanço anualInventário completo
